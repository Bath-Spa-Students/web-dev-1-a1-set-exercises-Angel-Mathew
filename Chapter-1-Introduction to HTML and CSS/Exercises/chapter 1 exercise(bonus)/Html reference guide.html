HTML

HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
It defines the content and structure of web content.
It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages
such as JavaScript.
To design a document elements and each element has purpose.
The elements are coded with pair of tags opening and closing tag some of the elements
there are no closing tag and there are knows void or empty elements.
They do not have content, used to support the content

Elements of HTML

  • !DocType html tag: to define the verson of the html
  • html tag: to specify the end and beginning of the
  • head tag:information that describes the web page document
  • body tag:Contain text and elements that display in the web page document
  • title tag:display name in the browser's title tab.Define title of the web page
  • h1 tag:h1 defines the heading there are 6 h tag.
    Instead of sub-heading 6 h tag are given to use as main heading and sub heading
  • p tag:p tag defines the paragraph.Is used for content or text of the document
  • b tag:b tag defines bold.Is used to make text bold
  • u tag:u tag defines underline.Is used for underlining the text
  • blockquote tag:start a block of the text for special emphasis
  • ul tag: ul tag defines unoderlist.Used for bullet or list making
  • li:contains an item in the list.
  • ol:ol tag defines orderlist.Used for a numbering or lettering system
  • div tag: block area of division on a web page with empty space above and below

Css

Css Cascading Style Sheets is a programming language used for specify the presentation and styling of the document for any markup language one of them is HTML.
It is designed to enable the content and presentation like layout,background color,font. It is improve information accessibility.
